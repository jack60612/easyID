# EasyID
 Facial Recognition app based on CompreFace

## Installation:
1. Install Git + Python
2. Clone this repository
3. Install dependencies `pip install -r requirements.txt`
4. Edit the config in `easyID/settings.py`
5. Run `python -m easyid`
